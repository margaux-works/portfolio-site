My Flix Angular

GitHub - Live Site

The MyFlix Angular Application is a single-page, responsive movie app that allows users to explore a vast database of movies, view detailed information about each, and manage a personalized user profile. This project, developed using Angular and TypeScript, demonstrates a robust combination of client-side and server-side development with a focus on user experience.

Tech Stack

Angular, TypeScript, Angular Material, Node.js, Express, MongoDB, JSDoc

Purpose and Context

This project was a key milestone in my web development journey during the CareerFoundry bootcamp. Building on a backend and database I had previously created for a React-based MyFlix app, I chose Angular for this implementation to expand my skills in modern frontend frameworks and TypeScript.

Objective

The goal was to create a full-featured frontend using Angular, showcasing a professional-level understanding of UI frameworks, state management, and API integrations. It was also an opportunity to apply a clean, modular architecture suitable for scalable projects

User Authentication

Secure registration and login functionality with JWT-based authentication.

Browse Movies

A responsive movie list view, featuring cards with movie posters and summary information.

Movie Details

Detailed views for movies, including modals for director and genre details.

User Profile

Update personal data through an intuitive form. Manage a list of favorite movies, with the ability to add or remove them. Delete the user account directly from the profile page.

User profile Management

The user profile page offers functionality to update user details, manage favorite movies, and delete the account. Reusing the modular components and forms made it easy to integrate validation and responsiveness.

Dynamic User Profile

Update The user-profile.component.ts file demonstrates how the user profile can be dynamically updated using an API call. It captures changes in the user data and communicates with the backend to persist updates, providing immediate feedback to the user with snackbars for success or error messages.

Code Highlight

The editUser method showcases how user data is submitted via an API request and how the UI is updated accordingly, including refreshing favorite movies and updating local storage.

Conclusion

The MyFlix Angular app represents a significant step in my web development journey! It allowed me to master Angular and TypeScript while reinforcing concepts of modular design and responsive UI development. I am proud of the polished, user-centric experience I was able to deliver and am eager to continue refining my skills in modern web technologies
